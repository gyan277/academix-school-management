## What Teachers Can Do

### ✅ 1. Mark Attendance
**Location:** Attendance page (opens automatically after login)

**Steps:**
1. Select your class from dropdown
2. Select today's date
3. Mark each student as:
   - **Present** ✓
   - **Absent** ✗
   - **Late** ⏰
4. Click "Save Attendance"

**Features:**
- See attendance summary (Total, Present, Absent, Late)
- View attendance percentage
- Access attendance history

---

### ✅ 2. Enter Exam Grades
**Location:** Academic Engine page

**Steps:**
1. Click "Academic Engine" in sidebar
2. Select your class (P1, P2, JHS1, or JHS2)
3. Select subject
4. Choose grading period (Mid-term or End-of-term)
5. Enter scores for each student:
   - **Class Score:** 0-30 points (30%)
   - **Exam Score:** 0-70 points (70%)
6. System automatically calculates:
   - Total mark (out of 100)
   - Grade (A1, A2, B1, etc.)
7. Click "Generate Reports" when done

**Grading Scale:**
- A1: 80-100
- A2: 75-79
- B1: 70-74
- B2: 65-69
- B3: 60-64
- C1: 55-59
- C2: 50-54
- C3: 45-49

## What Teachers CANNOT Do

### ❌ Restricted Access
- **Dashboard** - Admin only
- **Registrar** - Cannot add/edit students or staff
- **Communication** - Cannot send bulk SMS
- **Reports** - Cannot view system-wide reports
- **Settings** - Cannot change system settings

### Why These Restrictions?
- **Security:** Protects sensitive student data
- **Responsibility:** Each role has specific duties
- **Accountability:** Clear separation of responsibilities
- **Data Integrity:** Prevents unauthorized changes

## Tips for Teachers

### Attendance
✅ **Do:**
- Mark attendance daily
- Save before leaving the page
- Check attendance summary for accuracy
- Review attendance history regularly

❌ **Don't:**
- Forget to save after marking
- Mark attendance for other classes
- Change past attendance without admin approval

### Grades
✅ **Do:**
- Double-check scores before saving
- Enter all scores for the term
- Generate reports after completing all entries
- Add meaningful remarks for students

❌ **Don't:**
- Enter scores above maximum (30 for class, 70 for exam)
- Submit incomplete grade sheets
- Modify grades after report generation without admin

## Common Questions

### Q: Can I see other teachers' classes?
**A:** No, you can only access your assigned classes.

### Q: Can I edit student information?
**A:** No, only admin and registrar can edit student records.

### Q: Can I send messages to parents?
**A:** No, only admin can send bulk SMS. Contact admin for communication needs.

### Q: What if I make a mistake in grades?
**A:** Contact the administrator to correct submitted grades.

### Q: Can I view reports?
**A:** You can generate reports for your students, but cannot access system-wide analytics.

### Q: How do I get more classes assigned?
**A:** Contact the administrator to update your class assignments.

## Best Practices

### Daily Routine
1. **Morning:** Mark attendance for all your classes
2. **During Class:** Note any late arrivals
3. **End of Day:** Review and save all attendance
4. **Weekly:** Check attendance patterns
5. **End of Term:** Enter all grades and generate reports

### Grade Entry
1. **Collect:** Gather all assessment scores
2. **Verify:** Double-check calculations
3. **Enter:** Input scores carefully
4. **Review:** Check for errors
5. **Generate:** Create student reports
6. **Remarks:** Add personalized comments
